## Identidade visual obrigatória
- **Cores primárias:** Preto (#0A0A0A) para fundo geral; dourado (#F5B700) para elementos de ação, gráficos, bordas ativas, badges de recorde.
- **Cores secundárias:** Cinza escuro (#1A1A1A) para cards; cinza médio (#2A2A2A) para hover; texto branco (#FFFFFF) e cinza claro (#B0B0B0) para subtítulos.
- **Tipografia:** Inter ou Montserrat, pesos bold para títulos, regular para corpo.
- **Ícones:** Lucide React, na cor dourada quando representam ações importantes (ex: troféu, coroa, fogo).
- **Efeitos:** Bordas arredondadas (border-radius: 12px), sombras sutis em cards, botões com gradiente dourado ou borda dourada.
- **Avatar do atleta:** usar um placeholder com iniciais "IR" dentro de um círculo dourado ou uma silhueta de powerlifter.

- **Responsividade:** mobile-first (largura máxima 430px, centralizado em desktop). Fundo preto em toda a aplicação.
- **Alerta simulado:** todos os botões que exigiriam backend real (upload, análise IA, salvamento em nuvem) devem exibir `alert("Funcionalidade em desenvolvimento – versão demonstrativa Ironside")`. Exceto comentários e registros de dor (salvos localmente).
- **Performance:** sem erros no console, carregamento rápido.
